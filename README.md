## Repository Overview
This repository contains beginner-to-intermediate Python projects completed during a 2-month internship at MotionCut. Each project is a standalone script meant to be clear, educational, and easy to run.

## Projects included
1. `emoji_mood_responder.py` - Interactive mood responder with emojis and messages.
2. `fake_job_title_generator.py` - Random funny/creative job title generator.
3. `festival_reminder_bot.py` - Simple festival reminder using dates.
4. `auto_nickname_combiner.py` - Generates nicknames from given names.
5. `color_outfit_recommender.py` - Recommends outfit colors based on mood.
6. `bill_splitter.py` - Splits a bill among a group with validation.
7. `word_frequency_finder.py` - Counts word frequency in a paragraph.

## How to use
1. Clone or download this repository.
2. (Optional) Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # on Windows use `venv\Scripts\activate`
   ```
3. Install optional dependencies (none required for basic usage). If you want desktop notifications for the festival bot, install:
   ```bash
   pip install plyer
   ```
4. Run any script:
   ```bash
   python emoji_mood_responder.py
   ```
